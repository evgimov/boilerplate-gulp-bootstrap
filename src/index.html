boilerplate with Gulp and Bootstrap

To get you started the files that are built through gulp follow this organization:

					boilerplate-gulp-bootstrap
					| -- assets/
					|    | -- css/
					|    |    | -- normalize.css
					|    |    | -- bootstrap.css
					|    |    | -- bootstrap-theme.css
					|    |    | -- font-awesome.css
					|    |    | -- project-specific css
					|    | -- fonts/
					|    |    | -- any bootstrap fonts
					|    |    | -- any font-awesome fonts
					|    |    | -- project-specific fonts
					|    | -- js/
					|    |    | -- jquery.js
					|    |    | -- bootstrap.js
					|    |    | -- project-specific js
					| -- favicon.ico
					| -- index.html

where vendor assets are injected before project specific assets.
